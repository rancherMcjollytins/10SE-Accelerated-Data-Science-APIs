# 10SE-Accelerated-Data-Science-APIs
School task about APIs and abstraction and modules and what not.

### Features
This program allows users to interact with the FreeWeatherAPI to exchange data with a database and display current and forecast data in a command line.

### Requirements
This program requires:
- API Key
- Python
- requests   
- json
- matplotlib
- Internet connection
- FreeWeatherAPI (Active)

### Requirements.txt
To use requirements.txt and install packages faster, run:

```bash
pip install -r requirements.txt
```
into a command terminal (Use a virtual environment for best practice).

### API Key
You will need the API Key in a file called apikeys.txt.
I have the keys in a personal file, and will need to give it to you.
