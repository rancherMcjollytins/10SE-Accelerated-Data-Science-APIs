# 10SE-Accelerated-Data-Science-APIs
School task about APIs and abstraction and modules and what not.

### Requirements.txt
To use requirements.txt and install packages faster, run:

```bash
pip install -r requirements.txt
```
into a command terminal (Use a virtual environment for best practice).

You will need the API Key in a file called apikeys.txt.
